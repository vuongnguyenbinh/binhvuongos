# Layer 2: Backend + Database

## Overview

Connect static UI (Layer 1) to PostgreSQL database. 3 subsystems: DB setup + migrations, JWT auth, CRUD operations replacing hardcoded data. Plus JSON API for external tools (extensions, bots).

## Architecture

```
Browser в”Җв”ҖвҶ’ Fiber в”Җв”Җв”¬в”Җв”ҖвҶ’ templ (HTML SSR) в”Җв”ҖвҶ’ PostgreSQL (103.97.125.131)

Extension/Bot в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
                    в”Ӯ
            JWT cookie (UI) / API key (bots)
```

## Tech Stack

| Component | Choice |
|-----------|--------|
| DB | PostgreSQL 16 @ 103.97.125.131 |
| Migrations | golang-migrate (SQL files) |
| Query | sqlc вҶ’ type-safe generated Go |
| Driver | pgx v5 + pgxpool |
